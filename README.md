# Hi there, I'm Gil Huynh Long (Huỳnh Long Gil)👋

[![Unity](https://img.shields.io/badge/Unity-100000?style=flat&logo=unity&logoColor=white)](#)

I am a Game Developer with a background in Computer Science, currently focused on pushing the boundaries of indie game development. I am the founder of **Long Gil Studio**, where I'm building **GilongWorld**—an upcoming **AAA Indie Game**, featuring Monster Taming, Base Building, and Alchemy mechanics.
